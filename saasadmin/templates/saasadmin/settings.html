Access Control Notes

The /saasadmin/ area is restricted to superuser accounts only. Tenants access their own admin area at /admin/, which is protected by staff-level credentials.

To create a superuser from the command line:

pipenv run python manage.py shell -c "
from users.models import User
User.objects.create_superuser(
    email='admin@dc.com.pg',
    password='YOUR_PASSWORD'
)"
